Document Preparation
LaTeX is a high-quality typesetting system; it includes features designed for the production of technical and scientific documentation. LaTeX is the de facto standard for the communication and publication of scientific documents. LaTeX is able to typeset virtually anything and is simple to learn. On Linux, Kile and KBibTeX provide a complete solution for writing papers with LaTeX. On Windows, Miketex is often suggested.

Most people seem to learn LaTeX by using someone else's similar document as a template. I'll post some of my documents here as examples for others to use.

Citation Management
BibTeX is integrated with any LaTeX distribution and provides the in-text citation and bibliography management. Like LaTeX source files, the database is a simple text file that can be edited by hand, but I prefer to use JabRef, a graphical reference manager. JabRef integrates searching for literature via several databases and manages all the associated pdf files and URLs. A plug-in for JabRef integrates with OpenOffice so you can use the same database for LaTeX documents and standard word-processor files. Virtually any reference manager and web based literature database can "export" to a BibTeX database, including Zotero, Google Scholar, Refworks, PubMed, etc.
