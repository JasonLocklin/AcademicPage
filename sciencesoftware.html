Free Software in Science

Scientists often have extremely specialized needs, so software used in science is often used by small numbers of people. Free Software designed specifically for science is often designed with power, flexibility, and reliability in mind -and often lacks "point-and-click" interfaces. Fortunately, the power and speed advantages of re-usable syntax, and the unique benifits of community development, more than make up for the longer learning curve.

As a researcher in Behavioural Neuroscience, the following software list is severely biased to my needs. For software applicable to other fields, consider skimming Debian Science.

Data Collection
Python: a dynamic object-oriented programming language that can be used for many kinds of software development. It offers strong support for integration with other languages and tools, comes with extensive standard libraries, and can be learn-ed in a few days.

There are three main Python libraries for creating psychology experiments:
PyEPL:a library for coding psychology experiments in Python. It supports presentation of both visual and auditory stimuli, and supports both manual (keyboard/joystick) and sound (microphone) input as responses.

Psychopy: an open-source package for creating psychology and psycho-physics stimuli in Python. PsychoPy combines the graphical strengths of OpenGL with the easy Python syntax to give psycho-physics a free and simple stimulus presentation and control package.

Vision Egg:a powerful, flexible, and free way to produce stimuli for vision research experiments.

Data Processing
(G)awk: working with data files often requires making changes to various text files based on certain patterns, or extracting data from parts of certain lines while discarding the rest. To write a program to do this in a language such as C or Pascal is a time-consuming inconvenience that may take many lines of code. The job is easy and often requires one line with awk, especially the GNU implementation: gawk.

Data Analysis/Visualization
R project: R is a free software environment for statistical computing and graphics. A very basic GUI (similar to SPSS) for R is R Commander. On linux, try Rkward as a GUI/DE. Consider reading Using R for Psychological Research, by William Revelle, or R for SAS and SPSS Users, by Bob Muenchen, for those farmiliar with SPSS syntax.

Scipy: A python library for scientific computing (think matlab).

Octave: A programming language for numerical computations that is mostly compatible with matlab source.

Manuscript Writing
LaTeX:a high-quality typesetting system; it includes features designed for the production of technical and scientific documentation. LaTeX is the de facto standard for the communication and publication of scientific documents. LaTeX is able to typeset virtually anything and is simple to learn. On Linux, Kile and KBibTeX provide a complete solution for writing papers with LaTeX. On Windows, Miketex is often suggested.
